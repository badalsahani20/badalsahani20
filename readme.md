## 🚀 Featured Project

### [Notesify](https://github.com/badalsahani20/fullstack-notes-app) — AI-Powered Notes App

> A production-grade full-stack notes application with multi-provider AI orchestration, optimistic concurrency control, and banking-level auth patterns.

**What's under the hood:**
- 🤖 **Multi-provider AI routing** — Groq / NVIDIA / Gemini with cascading fallbacks
- 🔒 **OCC** — version fields on all documents, 409 conflict resolution, graceful state merging
- ⚡ **Sub-10ms response times** via Upstash Redis with self-healing cache invalidation
- 🛡️ **Production auth** — refresh token rotation, replay attack detection, 5-device session capping
- 🔍 **Hybrid search** — MongoDB full-text indexing + regex fallback (zero zero-results)
- 📉 **LCP 3s → 1.7s, CLS 0.29 → 0.04** after diagnosing 4 performance bottlenecks
